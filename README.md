# Hackday Scoreboard

Placar de pull requests feito para hackdays

### Instalando

Clone o repositório e instale as dependências.

```
yarn install
```

Preencha o arquivo `userList.js` dentro de `/src/constants`.

Rode o comando para iniciar o ambiente local:

```
yarn start
```
